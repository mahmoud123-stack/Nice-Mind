Nice Mind – Windows version v2.2

July 24, 2023

  • 1. Fix the problem that the QR code cannot be displayed normally during WiFi Transfer
  • 2. Fix the problem of losing focus when deleting text in outline mode
  • 3. Fix the problem that the picture cannot be inserted normally
  • 4. Optimize the interactive experience.
  • 5. Fix some known bugs

Nice Mind – Android version v7.9

July 10, 2023

  • 1. Add the Doc section to support the creation of outline notes
  • 2. The Doc section also supports synchronization to Google drive and Onedrive
  • 3. Optimize the resolution when exporting large maps to PNG
  • 4. Optimize some known bugs

Nice Mind – Windows version v2.1

July 6, 2023

  • 1. Add Doc section
  • 2. Fix the problem that floating themes will overlap
  • 3. Optimize the resolution of exporting large maps to PNG
  • 4. Can share works via lulj999888@163.com
  • 5. Optimize the interactive experience, add scroll bars to adjust the canvas size, overview, and statistical nodes on the edit page
  • 6. Optimize the interface
  • 7. Fix the problem that it cannot run on a computer without a C++ environment

Nice Mind – Android version v7.8

June 18, 2023

  • 1. Optimize the map performance of Android 13 and fix the stuck problem
  • 2. Avoid overlapping when adding floating topic
  • 3. Fix some known bugs

Nice Mind – Android version v7.7

July 12, 2023

  • 1. Two-way layout child nodes support switching between table layout and tree layout
  • 2. Fix the problem that the color setting in outline mode is not saved
  • 3. Fix the problem that the picture cannot be inserted normally
  • 4. Fix some known bugs.

Nice Mind – Windows version v2.0

July 24, 2023

  • 1. Outline mode supports exporting PNG and PDF
  • 2. The picture in outline mode can be stretched, and supports displaying multiple pictures in one line
  • 3. Outline mode supports svg
  • 4. Optimize the picture viewing control, support copying the inserted picture and save the picture as a file
  • 5. Adds a multi-selection file function, which supports exporting, moving, deleting, and copying together after selecting files in batches
  • 4. Fix some known bugs
  • 4. Add clipart resources
